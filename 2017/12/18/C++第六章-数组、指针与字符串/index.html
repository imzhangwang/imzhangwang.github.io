C++第六章 数组、指针与字符串

发表于 2017-12-18 | 分类于 c++

本文内容来源于[1] 郑莉、董渊、何江舟 C++语言程序设计(第4版)．北京：清华大学出版社，2010.7

1.数组的声明和使用

数组的声明包括的方面：

  1. 确定数组名称
  2. 确定数组元素类型
  3. 确定数组的结构（维数）

声明一维数组：数据类型 数组名 [常量表达式]；
例如：一维数组int b[10];
二维数组int a[5][3];

数组使用：数组名[下标表达式1]
注意：数组下标必须是整数，而且不能越界。
数组存储是顺序连续存储的，下标从0开始。

数组的初始化：

  • int a[3] = {1,1,1};声明数组时给出数组元素初值
  • int a[] = {1,1,1};列出全部元素的初值，可以不用说明元素个数。
  • float fa[5] = {1.0, 2.0, 3.0};可以只对数组中的部分元素进行初始化

2.数组作为函数参数

使用数组名传递数据时，传递的是地址。
在被调函数中对形参数组值进行改变，主调函数中实参数组的相应元素值也会改变。

把数组作为参数时一般不指定数组的第一维的大小，即使指定，也会被忽略。

4.指针

地址理解：假设每个学生住一间房，每个学生就相当于一个变量的内容，房间是存储单元，房号就是存储地址。
指针也是一种数据类型，具有指针类型的变量称为指针变量。指针变量是用于存放内存单元地址的。
声明指针语法形式：数据类型 *名称；其中*在这里表示声明的是一个指针类型的变量，数据类型可以任意，名称自己取。
例如int *ptr;。表示定义了一个指向int型数据的指针变量，这个指针的名称ptr，专门用来存放int型数据的地址。

与地址相关的运算“*”和“&”

*:指针运算符,*ptr表示ptr所指向的int型数据的值。
&:取地址运算符,&i表示获取变量i的存储单元地址。

指向常量的指针：不能通过指针来改变所指对象的值，但指针本身可以改变，可以指向另外的对象。

声明指针类型的常量，这时指针本身的值不能被改变。

一般情况指针的值只能赋值给相同类型的指针，但有一种特殊的void类型指针，可以存储任何类型的对象地址，就是说任何类型的指针都可以赋值给void类型的指针变量。经过使用类型显示转换，通过void类型的指针便可以访问任何类型的数据。

5.动态内存分配

运算符new的功能是动态分配内存，语法形式new 数据类型(初始化参数列表);

动态分配了用于存放int类型数据的内存空间，并将初值2存入该空间中，然后将首地址赋给指针point。

运算符delete用来删除由new建立的对象，释放指针所指向的内存空间,格式delete 指针名;。
若删除的是对象，该对象的析构函数将被调用。对于用new建立的对象，只能使用delete进行一次删除操作，如果对同一内存空间多次使用delete进行删除将会导致运行错误。

注意：用new分配的内存，必须用delete加以释放，否则会导致动态分配的内存无法回收，使得程序占据的内存越来越大，这叫做“内存泄露”。

new运算符动态创建一维数组语法：new 类型名[数组长度];。

若是使用new建立的数组，用delete删除时在指针名前面要加[],格式delete [] 指针名；。

6.用vector创建数组对象

形式vector<元素类型>数组对象名(数组长度);

注意：用vector定义的数组对象的所有元素都会被初始化，若数组元素为基本数据类型，所有元素会被初始化为0,若为类类型，则调用默认构造函数初始化。
初值也可以自己指定，形式：vector<元素类型>数组对象名(数组长度,元素初值);

vector数组对象的名字表示的就是一个数组对象，而非数组的首地址。
vector定义的数组对象具有一个重要的成员函数size(),它会返回数组的大小。
