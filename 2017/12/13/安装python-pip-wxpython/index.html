安装python pip wxpython

发表于 2017-12-13 | 分类于 python

1.下载安装python

python下载地址：
https://www.python.org/downloads/

2.安装pip

  1. 下载pip压缩包,下载地址：https://pypi.python.org/pypi/pip#downloads
  2. 解压下载下来的压缩包
  3. cmd进入到解压目录
  4. 使用命令python setup.py install安装pip
  5. 设置环境变量C:\Python34\Scripts;,也就是python的安装路径加Scripts
    测试pip是否安装成功：pip list
    注意：设置pip环境变量时格式：你的python安装路径\Scripts;

3.安装wxpython模块

在cmd中使用命令：pip install wxpython
