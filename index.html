本文内容来源于[1] 郑莉、董渊、何江舟 C++语言程序设计(第4版)．北京：清华大学出版社，2010.7

1.数组的声明和使用

数组的声明包括的方面：

  1. 确定数组名称
  2. 确定数组元素类型
  3. 确定数组的结构（维数）

声明一维数组：数据类型 数组名 [常量表达式]；
例如：一维数组int b[10];
二维数组int a[5][3];

数组使用：数组名[下标表达式1]
注意：数组下标必须是整数，而且不能越界。
数组存储是顺序连续存储的，下标从0开始。

数组的初始化：

  • int a[3] = {1,1,1};声明数组时给出数组元素初值
  • int a[] = {1,1,1};列出全部元素的初值，可以不用说明元素个数。
  • float fa[5] = {1.0, 2.0, 3.0};可以只对数组中的部分元素进行初始化

安装python pip wxpython

发表于 2017-12-13 | 分类于 python

1.下载安装python

python下载地址：
https://www.python.org/downloads/

2.安装pip

  1. 下载pip压缩包,下载地址：https://pypi.python.org/pypi/pip#downloads
  2. 解压下载下来的压缩包
  3. cmd进入到解压目录
  4. 使用命令python setup.py install安装pip
  5. 设置环境变量C:\Python34\Scripts;,也就是python的安装路径加Scripts
    测试pip是否安装成功：pip list
    注意：设置pip环境变量时格式：你的python安装路径\Scripts;

3.安装wxpython模块

在cmd中使用命令：pip install wxpython
